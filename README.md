# JobTrail
Web app for tracking job applications

<br/>

# Tech Stack
<br/>

## Frontend
 - Next.js
 - Radix UI
 - Tanstack Query
 - TailwindCSS
 - Cypress
 - Vercel (Deployment)

<br/>

## Backend
 - Bun
 - Cloudflare R2 & Amazon S3
 - ElysiaJS
 - BullMQ
 - PostgreSQL via Neon
 - DrizzleORM
 - Redis via Upstash
 - AI SDK
 - Railway (Deployment)
