# JobTrail
Web app for tracking job applications

# Tech Stack

 ## Frontend

 - Next.js
 - Radix UI
 - Tanstack Query
 - TailwindCSS
 - Cypress
 - Vercel (Deployment)

 ## Backend
 - Bun
 - Cloudflare R2 & Amazon S3
 - ElysiaJS
 - BullMQ
 - PostgreSQL via Neon
 - DrizzleORM
 - Redis via Upstash
 - AI SDK
 - Railway (Deployment)
